Тема

Электрические нагрузки

Ключевые слова

Полная мощность
произведение действующего значения напряжения на действующее значение тока в цепи переменного тока.

Активная мощность
полезная часть полной мощности, которая определяет прямое преобразование электрической энергии в другие необходимые виды энергии.

Реактивная мощность
часть полной мощности, определяемая электромагнитными полями, образующимися в процессе работы приборов. Реактивная мощность определяется характером нагрузки.

Ток
направленное движение электрически заряженных частиц под воздействием электрического поля.

Напряжение
отношение работы, выполненной электрическим полем для перенесения заряда, к значению переносимого заряда на полосе цепи.

Электрические нагрузки
величина потребления мощности различными приемниками или потребителями энергии.

Активная нагрузка
такой тип нагрузки, при котором происходит преобразование электроэнергии в тепловую энергию.

Индуктивная нагрузка
такой тип нагрузки, при котором происходит преобразование электроэнергии в энергию магнитного поля.

Емкостная нагрузка
такой тип нагрузки, при котором происходит преобразование электроэнергии в энергию электрического поля.

Активно – индуктивная нагрузка
такой тип нагрузки, при котором происходит преобразование электроэнергии в энергию теплового и магнитного полей.

Активно - емкостная нагрузка
такой тип нагрузки, при котором происходит преобразование электроэнергии в энергию теплового и электрического полей.

Векторная диаграмма
графическое изображение меняющихся по гармоническому закону величин и соотношений между ними при помощи векторов.

График электрических нагрузок
это графическое изображение расходуемой мощности за определенный отрезок времени.

Картограмма электрических нагрузок
генеральный план проектируемого объекта, на котором изображена картина распределения нагрузок потребителей электроэнергии по его подразделениям.

Рекомендации для обучающегося (самостоятельная работа)

  • Ознакомьтесь с теоретическим материалом по новой теме. Просмотрите динамическую инфографику по теме «Электрические нагрузки».
  • Если в процессе изучения возникнут вопросы, запишите их для дальнейшего обсуждения с преподавателем.
  • Обратите внимание на основные вопросы, подтверждающие взаимосвязь знаний и умений при изучении модуля:
    - Электрические нагрузки и в чем они измеряются;
    - Типы электрических нагрузок и их характерные электроприемники.
  • Возвращайтесь к наиболее сложным аспектам темы.
  • Соблюдайте здоровьесберегающий режим: чередуйте работу с электронными носителями с отдыхом и гимнастикой для глаз.

Рекомендации для обучающегося (самостоятельная работа)

  • Повторите теоретический материал по теме «Электрические нагрузки».
  • Запустите виртуальный тренажер по теме «Электрические нагрузки».
  • В ходе практической работы выполните тренировочные задания, которые помогут закрепить пройденный и освоить новый материал.
  • В процессе интерактивного взаимодействия с тренажером познакомьтесь с основными методами нагрузки электрических нагрузок, с картограммой электрических нагрузок и графиками электрических нагрузок.
  • Уделите особое внимание правильной организации рабочего места.
  • Проведите практическую работу с виртуальным оборудованием и фиксацией ее результатов в ЭОМ.
    При выполнении заданий обратите внимание на рекомендации преподавателя.
  • При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.

Рекомендации для обучающегося (самостоятельная работа)

  • Запустите интерактивную мини-игру по теме «Электрические нагрузки».
  • Последовательно ознакомьтесь с условиями учебной задачи.
  • Выполните задания мини-игры. Обратите внимание на отсутствие подсказок и ограниченное число попыток.
    При выполнении заданий обратите внимание на рекомендации преподавателя.
  • Выполните тренировочные задания на проверку приобретенных знаний, умений и навыков:
    - интерактивное задание с выбором одного правильного ответа.
    Осуществите самопроверку и предварительную оценку.
  • Используйте технологию пошагового контроля по результату за каждое задание.
  • Сформулируйте конечный результат своей работы.

    При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.
