Электрические нагрузки

На главную Назад


Чтобы наглядного проиллюстрировать, как расчётные активные мощности подразделений предприятия распределены по его территории, на генплане промышленного предприятия строится картограмма электрических нагрузок. Для каждого подразделения на генплан наносятся окружности, визуализирующие их расчётные активные мощности. С этой целью площадь каждого круга выполняется пропорционально этой мощности.

Для одного подразделения нагрузки представляем следующим способом: окружности, изображенные сплошными линиями, иллюстрируют нагрузку 0,4 кВ, в свою очередь, окружности, изображенные штриховыми линиями, иллюстрируют нагрузку и 6(10) кВ. В цехах с обоими типами нагрузки строят две окружности, имеющие общий центр.

Координаты центров, построенных на картограмме окружностей, принимаются равными координатам геометрических центров прямоугольников или иных фигур, представляющихся собой планы подразделений

Для демонстрации доли мощности, приходящейся на освещение подразделений, внутри кругов, соответствующих нагрузке 0,4 кВ, выделяется заштрихованный сектор, площадь которого пропорциональна этой доле.
